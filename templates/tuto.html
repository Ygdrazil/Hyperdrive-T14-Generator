Tutoriel d'utilisation

Episode 0 : Introduction

Bienvenue sur le tutoriel d'utilisation de l'interface web T-14 Hyperdrive Generator.
Cette application vous permettra de créer de manière simple et efficace des parties de votre site internet, tel que des liens ou des boutons.
Après avoir suivi ce tutoriel vous saurez travaillez de manière efficace avec cette interface.

Episode 1 : Les options

Dès que vous saurez quel élément vous voulez dans votre site, vous pourrez cliquer sur ce que vous souhaitez dans la barre de navigation.

Afin de vous donner le plein pouvoir sur les élements, tel Palpatine qui maitrise les éclairs, vous avez accès à un panel d'option de personnalisation.

Selon l'élément que vous avez choisis, les options seront différentes.
