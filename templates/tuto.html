Tutoriel d'utilisation

Episode 0 : Introduction

Bienvenue sur le tutoriel d'utilisation de l'interface web T-14 Hyperdrive Generator.
Cette application vous permettra de créer de manière simple et efficace des parties de votre site internet, tel que des liens ou des boutons.
Après avoir suivi ce tutoriel vous saurez travailler de manière efficace avec cette interface.

Episode 1 : Les options

Dès que vous saurez quel élément vous voulez dans votre site, vous pourrez cliquer sur ce que vous souhaitez dans la barre de navigation.



Afin de vous donner le plein pouvoir sur les élements, tel Palpatine qui maitrise les éclairs, vous avez accès à un panel d'option de personnalisation.

Le panneau "option" vous permet de changer le style des éléments de nombreuses manière :

  • Changement du style du texte :
    • Police;
    • Taille;
    • Couleur;
    • Couleur du sulignage;
    • Douceur du surignage;
    • Formatage du texte : gras, sous-ligne, italique;
    • Les marges internes de l'élément, c'est à dire l'espace entre bord de l'élément et le texte;
    • Les marges externes de l'élément, c'est à dire l'espace entre bord de l'élément et la marge la plus proche;
  • Changement du comportement de l'élément au survol :
    • Temps de transition;
    • Element de style propre au texte;


Episode 2 : Le rendu et le code

Cette partie est assez courte :

Vous avez le rendu de votre demande à droite du tableau de commande (si vous êtes sur PC) ou en-dessous (si vous êtes sur téléphone).

Dans un second temps, le code vous sera donné et vous pourrez le récupérer afin d'enrichir votre site web.

Vous aurez en premier la balise HTML et ensuite le code CSS dans une balise style.

Vous savez désormais tout ce qu'il y a à savoir pour utiliser notre générateur à pleine puissance !!
